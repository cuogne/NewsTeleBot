## Bot Telegram

> Version rut gon crawl web cua con [bot discord](https://github.com/cuogne/discord-bot) va viet bang Golang.

## Tech Stack

- **Language:** Go (v1.26.1)
- **Database:** PostgreSQL (Supabase)
- **Crawler:** Colly (HTML), gofeed (RSS)
- **Handler:** go-readability
- **AI Summarization:** Gemini API
- **Telegram Bot:** Telebot (v4)
- **Scheduler:** robfig/cron
- **Concurrency:** Goroutines, Channels, sync.WaitGroup, Worker Pool
- **Deployment:** Docker, GitHub Actions CI/CD
